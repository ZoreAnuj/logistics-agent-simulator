# Logistics Agent Simulator

An experimental simulation platform for AI-driven logistics management. It explores how multi-step autonomous agents can optimize supply chain operations using real-time data and vector search.

## Key Features
- Simulates predictive maintenance and dynamic route optimization
- Integrates vector search for intelligent document and data retrieval
- Models multi-agent emergency response workflows
- Provides a sandbox for testing autonomous logistics decisions

## Tech Stack
- Python
- TiDB Serverless (Vector Search)
- AI Agent Frameworks

## Getting Started
```bash
git clone https://github.com/zoreanuj/logistics-agent-simulator.git
cd logistics-agent-simulator
pip install -r requirements.txt
# Configure environment variables and run the simulation
```
